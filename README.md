## What is this?
A simple website with login system.

## Project Detail

* Python 3
* Django 2.2.3
* Bootstrap 4

## Run it in local env

```bash
git clone https://github.com/ZFCon/arablinks.git
cd moviesdand
git checkout master

# setup virtualenv
pip install -r requirements.txt

./manage.py runserver
# http://127.0.0.1:8000/
```
